# 💾 Local Data Storage

Your data stays on your device.

### How It Works

* Data is stored locally in your browser
* No cloud servers
* No external database
* No account required
* Privacy-first approach

### Persistence

✅ Browser refresh → Data remains

✅ Browser close/reopen → Data remains

# 🔄 Backup & Restore

Never lose your progress.

### Export

Save all your data as:

```json
habitsentia.json
```

### Import

Restore your previously exported file and continue exactly where you left off.

### Recommended Practice

Create backups regularly:

```text
Daily
Weekly
Before Browser Cleanup
Before System Reset
```
